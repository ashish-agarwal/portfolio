# Project Portfolio #
## Getting Started
It is a project which showcases my projects. This is built using html , css and Javascript.

## Prerequisites
    * Grunt 
    * GraphicsMagick


## Usage

Clone the repository
```
git clone https://github.com/ashish-agarwal/portfolio.git
```

Install bower components

```
bower install
```
